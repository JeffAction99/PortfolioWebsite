The Only Way is DOWN!

Description

This is a game that I made for Brackeys Game Jam 2023.2. It was the first game jam I ever took part in. The theme was "Dive Deeper", so I made the game so that there are three levels about going deeper into the Earth. Specifically, overground, underground, and within the Earth's core. I also gave these three levels different difficulty options.

The game is a 2D platformer, to where the player runs, jumps and falls down different platforms. It is a very simple game to get into. The challenge is that some platforms below you cannot see and you're hoping that you land on one, because if you fall for too long, you go back to the beginning of the level.

After the game was made, I got some feedback about it. It was a good start from a prototype standpoint. One of the more common pieces of feedback I got was about the level structure and how the player was falling blindly hoping that they land on a platform. Although, it was intentional for a more challenging experience, I learnt that maybe this wasn't a good idea. I am currently developing an improved version based on the feedback I received.
